Full CI/CD pipeline on an open source banking portal (the source is in each folder), using github action to build image the source code, ArgoCD deploy to EKS with prometheus/grafana monitoring the detail metrics in each nodes, Cloudwatch to monitor overall of the cluster, using the RDS as main database from Amazon.
